# 康威生命游戏 | 元胞自动机模拟器

一个基于HTML5 Canvas的康威生命游戏模拟器，支持绘制、导入导出、缩放和平移等功能。

## 功能特点

### 基础功能
- **网格绘制**：支持鼠标绘制/擦除细胞，点击切换细胞状态
- **文件导入**：支持导入RLE格式或自定义JSON格式的生命游戏模式文件
- **模拟控制**：开始/暂停模拟，调节演算速度（0.1x-10x）
- **视图控制**：支持画布缩放和平移，适应不同大小的模式

### 亮点功能
- **模式库**：内置经典生命游戏模式（如滑翔机、脉冲星等）
- **状态保存**：可保存当前网格状态为文件

## 如何使用

### 本地运行
1. 克隆或下载本仓库
2. 直接在浏览器中打开 `index.html` 文件

### 基本操作
- **绘制细胞**：选择"绘制"工具，在网格上点击或拖拽
- **擦除细胞**：选择"擦除"工具，在网格上点击或拖拽
- **运行模拟**：点击"开始"按钮，再次点击暂停
- **调整速度**：使用速度滑块调节模拟速度
- **缩放视图**：使用鼠标滚轮或缩放按钮
- **平移视图**：点击并拖拽空白区域

### 文件操作
- **保存**：将当前状态保存为JSON文件
- **加载**：加载之前保存的JSON文件或RLE格式文件
- **导出**：将当前模式导出为RLE或JSON格式

## 技术实现

- **前端框架**：纯HTML5 + CSS + JavaScript
- **UI框架**：Tailwind CSS v3
- **图标库**：Font Awesome
- **图形渲染**：HTML5 Canvas
- **文件处理**：FileReader API

## 游戏规则

康威生命游戏遵循以下简单规则：
1. 存活细胞周围有2-3个存活细胞，保持存活
2. 存活细胞周围存活细胞少于2个或多于3个，死亡
3. 死亡细胞周围有3个存活细胞，变为存活
